# CIATEC UNALM - Semillero de Investigación en Agrotecnología 🛰️🌱

¡Bienvenidos al repositorio oficial del **CIATEC**! Somos un semillero de investigación de la Facultad de Agronomía de la Universidad Nacional Agraria La Molina, dedicado a integrar la tecnología y la ciencia de datos en el sector agropecuario.

## 🚀 Nuestra Misión
Cerrar la brecha entre el campo y la tecnología mediante el desarrollo de soluciones de hardware y software de código abierto. Nos enfocamos en la **Agricultura 4.0** para optimizar recursos y mejorar la resiliencia de los cultivos.

## 🛠️ Áreas de Desarrollo
* **Internet de las Cosas (IoT):** Diseño de sistemas de monitoreo con microcontroladores (ESP32) para variables críticas como humedad del suelo, temperatura y radiación.
* **Fisiología Vegetal:** Análisis del comportamiento de cultivos (como *Vigna unguiculata*) bajo condiciones de estrés hídrico.
* **Data Science Agronómico:** Procesamiento de datos y modelamiento estadístico utilizando R y Python.

## 📂 Estructura del Repositorio
Para mantener el orden en nuestras investigaciones, utilizamos la siguiente estructura de carpetas:

- `hardware/`: Firmwares (.ino, .cpp) y diagramas de circuitos.
- `scripts/`: Código de procesamiento de datos en R y Python.
- `data/`: Conjuntos de datos (crudos y procesados).
- `docs/`: Manuales, bitácoras de campo y documentación técnica.
